# Telegram Bot для аренды серверов

Telegram-бот для аренды личных серверов с админ-панелью и автоматической обработкой платежей.

## Концепция

**Важно:** Это не VPN-сервис, а аренда личных серверов.

Юридически это:
- Хостинг-услуга (аренда сервера)
- Пользователь сам настраивает прокси/туннель
- Не подпадает под регулирование VPN-провайдеров в РФ

## Возможности

✅ Продающая воронка через кнопки
✅ 4 тарифа аренды с разными периодами
✅ Перенаправление в личку @keml00
✅ Админ-панель для управления
✅ База данных SQLite
✅ Статистика и аналитика

## Установка

### 1. Установи Python 3.8+

Проверь версию:
```bash
python --version
```

### 2. Установи зависимости

```bash
pip install -r requirements.txt
```

### 3. Создай бота в Telegram

1. Найди [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot`
3. Придумай имя и username для бота
4. Скопируй токен (выглядит как `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 4. Узнай свой Telegram ID

1. Найди [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправь `/start`
3. Скопируй свой ID (число)

### 5. Настрой конфиг

Скопируй `.env.example` в `.env`:
```bash
cp .env.example .env
```

Открой `.env` и заполни:
```env
BOT_TOKEN=твой_токен_от_BotFather
ADMIN_ID=твой_telegram_id

SBER_CARD=2200 7007 1234 5678
YOOMONEY=410011234567890
CRYPTO_ADDRESS=твой_крипто_адрес

VPN_SERVER=твой.vpn.сервер.com
VPN_PORT=443
```

## Запуск

```bash
python bot.py
```

Если всё ок, увидишь:
```
INFO:__main__:База данных инициализирована
INFO:__main__:Бот запущен
```

И тебе придёт сообщение в Telegram: "🤖 Бот запущен!"

## Использование

### Для пользователей

1. Пользователь нажимает кнопку тарифа
2. Получает информацию о выбранном тарифе
3. Нажимает "Написать @keml00"
4. Открывается личка с предзаполненным текстом
5. Оформляет аренду напрямую с владельцем

### Для админа

**Команды:**
- `/admin` — статистика бота
- `/pending` — список ожидающих платежей
- `/status` — проверить свою подписку

**Подтверждение платежей:**
1. Тебе придёт уведомление со скриншотом
2. Нажми "✅ Подтвердить" или "❌ Отклонить"
3. При подтверждении пользователь автоматически получит доступ

## Интеграция с сервером

Сейчас бот генерирует заглушку ключа в [bot.py:234-238](bot.py#L234-L238):

```python
def generate_vpn_key() -> str:
    key_id = str(uuid.uuid4())
    return f"vless://{key_id}@{SERVER_HOST}:{SERVER_PORT}?encryption=none&security=tls&type=tcp"
```

**Для реальной работы:**
1. Подключи API своего сервера (V2Ray, Outline, Shadowsocks)
2. Замени функцию `generate_vpn_key()` на реальную генерацию ключей
3. Добавь управление пользователями на сервере

## Юридические аспекты

**Почему это легально:**
- Ты сдаёшь в аренду сервер (хостинг-услуга)
- Пользователь сам настраивает прокси/туннель
- Не требуется регистрация как VPN-провайдер
- Не нужно подключение к ФГИС РКН
- Не нужно хранить логи по закону Яровой

**Рекомендации:**
- В договоре/оферте указывай "аренда сервера"
- Не используй термин "VPN" в официальных документах
- Предоставляй инструкции по настройке, а не готовый VPN

## Troubleshooting

**Бот не запускается:**
- Проверь токен в `.env`
- Проверь что установлены все зависимости

**Не приходят уведомления админу:**
- Проверь `ADMIN_ID` в `.env`
- Убедись что это твой Telegram ID

**Ошибка базы данных:**
- Удали `vpn_bot.db` и перезапусти бота
